collective.dexterityhelpers Installation
----------------------------------------

To install collective.dexterityhelpers using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.dexterityhelpers`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.dexterityhelpers
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
